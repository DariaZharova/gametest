# Детективное агентство — Telegram бот

Интерактивная детективная игра в Telegram. Вы — следователь, который расследует убийство Максима Лебедева. Общайтесь со свидетелями, собирайте улики, допрашивайте подозреваемых. Каждый персонаж — ИИ с собственной линией лжи и триггерами раскрытия.

## Быстрый старт

1. Клонируйте репозиторий
2. Создайте виртуальное окружение, активируйте
3. Установите зависимости: `make install`
4. Скопируйте `.env.example` в `.env` и заполните токены
5. Инициализируйте БД: `make migrate`
6. Запустите бота: `make run`

## Структура

- `bot/` — основной код
- `bot/prompts/` — системные промпты для ИИ
- `bot/data/` — статические данные (локации, улики, персонажи)
- `docs/` — документация, канон дела

## Механика

Игрок пишет сообщения боту. Роутер определяет, кому они адресованы (NPC, напарник, перемещение) и вызывает соответствующего ИИ. Прогресс сохраняется в SQLite.

Подробнее — в `docs/case_bible.md`.
